Commands

Here is a list of commands that you can use with doremi. You can also copy the code for each command:

  • avatar [user] - Gets the profile/avatar of you or the mentioned user.

    module.exports = {
                    name: "avatar",
                    $if: "old",
                    code: `
                  $color[#E6E6FA]
                  $if[$mentioned[1]==]
                  $author[$username's Avatar!]
                  $image[$authorAvatar[2048]]
                  $else
                  $author[$username[$mentioned[1]]'s Avatar!]
                  $image[$userAvatar[$mentioned[1];2048]]
                  $endif
                  $suppressErrors[{newEmbed:{description:Something went wrong!}{color:#E6E6FA}}]
                  `,
                };

  • servericon - Gets the guild's icon.

    module.exports = {
                    name: "servericon",
                    code: `
                    $color[#E6E6FA]
                    $author[$guildName[$guildID]'s Icon!]
                    $image[$guildIcon[$guildID]]
                    `,
                  };

  • ping - Gets a response with the latency.

    client.command({
                    name: "ping",
                    code: 'Pong! $pingms'
                  });
